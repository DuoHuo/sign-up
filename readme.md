#报名介绍
============
```js
*首先将sign-up fork到自己git上
*再将其clone到本地
*在本地文件夹下填写个人信息
*将该文件夹push到git上
*向我发起pull request
```
##个人信息包括：
=========
```js
   *姓名
   *学院+专业
   *联系方式（微信或者QQ）
   *个人爱好
```
